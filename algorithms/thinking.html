Algorithm 03

算法思想

算法思想比具体题目更重要。它帮助你把陌生问题转换成熟悉模型。

递归与分治

递归关注终止条件和状态传递；分治强调把大问题拆成可合并的小问题。

  • 递归：树遍历、DFS、回溯的基础。
  • 分治：快速排序、归并排序、FFT 的共同思想。

动态规划

动态规划训练的是状态定义能力：状态、状态转移、最优子结构。

爬楼梯 背包问题 最长递增子序列 编辑距离 股票买卖 
def climb_stairs(n):
    if n <= 2:
        return n

    prev, cur = 1, 2
    for _ in range(3, n + 1):
        prev, cur = cur, prev + cur
    return cur

贪心与回溯

贪心需要证明局部最优能推导全局最优；回溯本质是暴力搜索加剪枝。

  • 贪心：区间问题、Huffman 编码、部分最小生成树思路。
  • 回溯：全排列、N 皇后、数独。
def permutations(nums):
    ans, path, used = [], [], [False] * len(nums)

    def dfs():
        if len(path) == len(nums):
            ans.append(path[:])
            return
        for i, value in enumerate(nums):
            if used[i]:
                continue
            used[i] = True
            path.append(value)
            dfs()
            path.pop()
            used[i] = False

    dfs()
    return ans
