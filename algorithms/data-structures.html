Algorithm 02

数据结构

数据结构决定数据如何组织，算法决定如何在这些结构上行动。

链表

链表适合练指针和引用思维。必须掌握反转、合并、快慢指针、环检测。

树

树是程序员最重要的数据结构之一。二叉树遍历、BST、堆、Trie 都很常见。

内容 用途
二叉树遍历 递归、迭代、层序遍历基础
BST 有序数据查找与维护
堆 优先队列、Top K、任务调度
Trie 前缀匹配、搜索提示、词典

图

图用于描述关系网络，常见于推荐、社交、地图、依赖管理和工作流。

  • BFS / DFS：搜索与遍历。
  • Dijkstra / Floyd：最短路径。
  • 拓扑排序：依赖管理。
  • 并查集：连通性与判环。
