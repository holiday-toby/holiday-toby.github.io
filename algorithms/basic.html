基础算法

排序、二分、哈希、双指针、栈与队列，是程序员解决问题时最先调用的工具箱。

排序

重点掌握快排、归并、堆排序，以及稳定排序和非稳定排序的差异。

  • 快速排序：平均 O(n log n)，工程里常见，核心是分区。
  • 归并排序：稳定，适合链表排序和外部排序。
  • 堆排序：适合 Top K 和优先级场景。
def quick_sort(nums):
    if len(nums) <= 1:
        return nums

    pivot = nums[len(nums) // 2]
    left = [x for x in nums if x < pivot]
    middle = [x for x in nums if x == pivot]
    right = [x for x in nums if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)

二分查找

二分不仅用于找元素，也用于找边界和“二分答案”。

  • 普通二分：在有序数组中查找目标值。
  • 左边界 / 右边界：处理重复元素和范围查询。
  • 二分答案：把最优解问题转成可判定问题。
def lower_bound(nums, target):
    left, right = 0, len(nums)
    while left < right:
        mid = (left + right) // 2
        if nums[mid] < target:
            left = mid + 1
        else:
            right = mid
    return left
