Algorithm 06

算法学习路线

不要乱刷题。先建立结构，再按题型训练，最后回到工程场景里验证。

第一阶段：基础

  1. 数组、链表、栈、队列。
  2. 哈希、二叉树、排序、二分。
  3. 用简单题训练边界条件和复杂度意识。

第二阶段：核心思想

  1. DFS / BFS。
  2. 递归、回溯、贪心。
  3. 动态规划，重点练状态设计。

第三阶段：进阶

  1. 图、并查集、Trie、堆。
  2. 线段树、位运算、状态压缩。
  3. 结合数据库、缓存、搜索、AI 检索理解工程应用。

路线图图示

def weekly_plan(week):
    stages = [
        ("基础", ["array", "linked list", "stack", "binary search"]),
        ("核心思想", ["dfs", "bfs", "backtracking", "dynamic programming"]),
        ("进阶工程", ["graph", "heap", "trie", "cache", "vector search"]),
    ]
    stage_name, topics = stages[min(week // 4, len(stages) - 1)]
    topic = topics[week % len(topics)]
    return f"第 {week + 1} 周：{stage_name} - {topic}"


for week in range(12):
    print(weekly_plan(week))

不同方向重点

  • 后端：Hash、堆、图、缓存、一致性哈希。
  • 前端：树、DFS/BFS、Diff 算法、字符串。
  • AI 工程：线代、概率、优化算法、向量搜索。
  • 游戏开发：图算法、A*、空间分区、碰撞检测。
