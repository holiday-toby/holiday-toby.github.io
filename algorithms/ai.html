向量检索

向量检索是 RAG、AI 搜索、推荐系统的重要基础。

  • Embedding：把文本、图片、用户行为表示成向量。
  • ANN：近似最近邻检索，用速度换取可接受的召回。
  • HNSW / FAISS：工程中常见的向量检索方案。
import math

def cosine_similarity(a, b):
    dot = sum(x * y for x, y in zip(a, b))
    norm_a = math.sqrt(sum(x * x for x in a))
    norm_b = math.sqrt(sum(y * y for y in b))
    return dot / (norm_a * norm_b)

def search(query_vec, docs, top_k=3):
    scored = [
        (cosine_similarity(query_vec, doc_vec), doc_id)
        for doc_id, doc_vec in docs.items()
    ]
    return sorted(scored, reverse=True)[:top_k]

机器学习基础

至少理解线性回归、决策树、XGBoost、KMeans 这些经典模型的目标和适用场景。

def predict_linear(x, weight, bias):
    return weight * x + bias

def mse_loss(samples, weight, bias):
    errors = [
        (predict_linear(x, weight, bias) - y) ** 2
        for x, y in samples
    ]
    return sum(errors) / len(errors)
